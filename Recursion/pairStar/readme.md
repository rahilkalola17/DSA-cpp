## Pair star
Given a string, compute recursively a new string where identical chars that are adjacent in the original string are separated from each other by a "*".
### Sample Input 1 :
hello
### Sample Output 1:
hel*lo
### Sample Input 2 :
xxyy
### Sample Output 2:
x*xy*y
### Sample Input 3 :
aaaa
### Sample Output 3:
a*a*a*a
